# Инструкция для работы с Markdown

## Выделение текста 

Чтобы выделить текст курсивом необходимо обрамить его зёздочками (*) или знаком нижнего подчёркивания (_)

*пример*

Чтобы выделить текст полужирным необходимо обрамить его двумя звёздочками (*****) или двойным знаком нижнего подчёркивания (____)

**пример**

Альтернативные способы выделения текста жирным или курсивом нужны для того, чтобы мы могли совмещать оба этих способа. _Например текст может быть выделен курсивом и при этом быть **полужирным**_

## Списки
Чтобы добавить ненумерованные списки необходимо пункты выделить звёздочкой

Чтобы добавить ненумерованные списки необходимо пункты выделить (*)

Например вот так: 
* Элемент 1

* Элемент 3

Чтобы добавить нумерованные списки необходимо их пронумеровать
1. Первый пункт

## Ссылки 
**Синтаксис ссылок в языке Markdown**

В языке Markdown существует два основных способа создания ссылок:

* **Ссылки с текстом**
* **Ссылки без текста**

**Ссылки с текстом**

Чтобы создать ссылку с текстом, необходимо заключить текст ссылки в квадратные скобки, а затем указать URL-адрес ссылки в круглых скобках. Например:

```markdown
[Текст ссылки](URL-адрес)
```

Пример:

```markdown
[Google](https://www.google.com)
```

**Ссылки без текста**

Чтобы создать ссылку без текста, необходимо заключить URL-адрес ссылки в квадратные скобки и добавить символ `!` перед URL-адресом. Например:

```markdown
![URL-адрес](URL-адрес)
```

Пример:

```markdown
![https://www.google.com](https://www.google.com)
```

**Дополнительные возможности**

* **Относительные ссылки**

Относительные ссылки ссылаются на другие документы в том же каталоге, что и текущий документ. Например, если текущий документ находится в каталоге `/docs`, а другой документ находится в подкаталоге `/examples`, то ссылка на этот документ будет выглядеть следующим образом:

```markdown
[Ссылка на документ](./examples/example.md)
```

* **Ссылки на изображения**

Чтобы создать ссылку на изображение, необходимо указать URL-адрес изображения в квадратных скобках. Например:

```markdown
[Изображение](https://example.com/image.png)
```

* **Ссылки на файлы**

Чтобы создать ссылку на файл, необходимо указать URL-адрес файла в квадратных скобках. Например:

```markdown
[Файл](https://example.com/file.txt)
```

* **Ссылки на электронную почту**

Чтобы создать ссылку на электронную почту, необходимо указать адрес электронной почты в квадратных скобках. Например:

```markdown
[Электронная почта](mailto:example@example.com)
```

* **Ссылки на социальные сети**

Чтобы создать ссылку на социальную сеть, необходимо указать URL-адрес страницы в социальной сети в квадратных скобках. Например:

```markdown
[Ссылка на Twitter](https://twitter.com/example)
```

**Примеры**

```markdown
[Google](https://www.google.com)

![https://www.google.com](https://www.google.com)

[Ссылка на документ](./examples/example.md)

[Изображение](https://example.com/image.png)

[Файл](https://example.com/file.txt)

[Электронная почта](mailto:example@example.com)

[Ссылка на Twitter](https://twitter.com/example)
```
